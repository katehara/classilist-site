About Classilist

Probabilistic Models involve the uncertainty in the modelling that causes it to less sensitivity to noise phenomenon and hence is widely used alongside traditional Machine Learning Techniques and Probabilistic Classification constitutes a major part of it.

Being a widely preferred field in predictive analytics, Graphicals Models to analyse the prababilistic classification data are a must.

Classilist is an Open-Source Visualization Dashboard, aimed towards Visual Analytics for Probabilistic Classification Data. Having been developed as a Google Summer of Code'16 Project, it is based on a High Quality Research Work done at CSE department of TUWien.

Classilist sports some really awesome features like - Class Probability Distribution, Feature View Analysis, support for RapidMiner, KNIME and R to ease data formatting and export which will be used by classilist and also support for WEKA will soon be introduced.

Installation
Documented and Video Instructions to install Classilist and its support components. Video tutorials for installation can be accessed by button at the beginning of every section.
  • Install Classilist Interface
    Download & Install
    • Download 'Classilist Interface' from Downloads & Links section
    • Extract the zipped file (classilist.zip) and copy into the desired directory
    • Now your Classilist Installation directory is [current-directory]/classilist
  • Install R Package
    Method 1 : Install from Source
    • Install Package 'devtools' [ How to install? ]
    • Load Package 'devtools'
      library(devtools)
    • Install and Load Package 'classilistR' from BitBucket User : typhon23 and Repo : classilistR
      install_bitbucket("typhon23/classilistR")
      library(classilistR)
    Method 2 : Download & Install
    • Download 'R support package' from Downloads & Links section
    • Copy the zipped file (classilistR.zip) into the current working directory.
      Path of Current Working Directory can be found by getwd()
    • Install and Load Package 'classilistR'
      install.packages("classilistR", repos = NULL)
      library(classilistR)
  • Install RapidMiner Plugin
    Download & Install
    • Download 'RapidMiner Operator Extension' from Downloads & Links section. If you are asked to 'keep' or 'discard' the .jar file, be sure to choose keep.
    • Copy the .jar file (classilistRM-1.0.0-all.jar) to the RapidMiner plugins folder. Path of RapidMiner plugins folder is : [rapidminer-studio-home]/lib/plugins
    • Start (Restart if already running) RapidMiner. In the operator tree, find Operator Extensions>Classilist>Classilist Group>Classilist Export
  • Install KNIME Plugin
    Download & Install
    • Download 'KNIME Node Plugin' from Downloads & Links section. If you are asked to 'keep' or 'discard' the .jar file, be sure to choose keep.
    • Copy the .jar file (org.classilist.knime_1.0.0.jar) to the KNIME dropins folder. Path of KNIME dropins folder is : [KNIME-home]/dropins
    • Start (Restart if already running) KNIME. In your node tree, find Node Classilist
    • This will also work on Eclipse Installation of KNIME.
  • Install WEKA Plugin
    Download & Install
    • Download 'WEKA Extension' from Downloads & Links section
    • Start WEKA, go to 'Tools' in Menu-bar, click on 'Package Manager'
    • Select 'File/URL' in top right corner.
    • Browse and select the downloaded zipped file (classilistWEKA.zip) and click 'Open/OK'.
    • You will find the installed extension in the form of menu extension.
    WEKA Plugin is in testing phase. It will be made available in the Downloads & Links section soon.
